Pacman is package manager.

Commands that i have used  -

1. `pacman -S <package>` 
	1. Installs the package specified.
2. `pacman -R <package>` 
	1. Removes the package specified.
3. `pacman -q`
	1. Used for query the installed packages.
4. `pacman -Syu`
	1. Combination of three commands
		A. `-S` for sync (install)
		B. `-y` for getting the latest from server
		C. `-u` for updating the packages to the latest version.
