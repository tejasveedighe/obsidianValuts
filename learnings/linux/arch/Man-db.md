- `Man-dd` and `man-pages` are packages that are used for caching and searching the Man pages.
- Man is short for Manual. So manual for linux OS.
- The tools allows for getting the man page for a particular command using the terminal.
- Using the command - `man pacman` 
	- Gets the man page for `pacman` or any other specified tool.
- The man pages are used for learning about tools and the linux OS that is installed.
